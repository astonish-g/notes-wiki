##### Info:
Int or integer is a whole number, **positive** or **negative**, **without decimals** 

###### Examples:

```Python
x = 1
y = 356382985829198382
z = -84297

print(type(x))
print(type(y))
print(type(z))
```

All the above`print()` **functions** will give **int** output as they are all integer numbers.
