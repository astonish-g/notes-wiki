##### Info:
In Python, when we are working with **strings**, the strings starts to get indexed by 0.

##### Example:
```Python
phrase = "Giraffe Academy"
          01234567
```
So on the example above you can see that G letter is indexed as 0, r letter is indexed as 2..etc
